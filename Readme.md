## 🛠️ Setup


1. Clone this repository:
   """bash
      git clone https://github.com/sushantmittal1205/sm-pytorch.git
         cd sm-pytorch
	    """

	    2. Install dependencies:
	       """bash
	          pip install torch torchvision torchaudio

		        """

			   """bash

			         """
